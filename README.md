# AI-Powered SEO Content Cluster Generator

## 📌 Project Overview

This project demonstrates how Prompt Engineering and AI can be used to create SEO-optimized content clusters for local businesses.

The goal is to generate a complete SEO content strategy, including a pillar blog, supporting blogs, keyword planning, internal linking, and local SEO optimization.

---

## 🏢 Business Chosen

**UrbanNest Interiors**

Location: Bangalore, India

Industry: Interior Design & Home Renovation

---

## 🎯 Objective

Build a reusable AI prompt system capable of generating:

- SEO-friendly blog outlines
- Long-form pillar blogs
- Supporting blog content
- Keyword strategy
- Internal linking suggestions
- Local SEO optimization
